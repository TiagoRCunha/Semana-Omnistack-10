# Semana Omnistack 10

para testar o backend

```
cd backend
yarn || npm install
yarn dev || npm run dev
```

para testar o frontend

```
cd frontend
yarn || npm install
yarn start || npm start
```

para o mobile

```
cd mobile
yarn || npm install

yarn start || npm start

for android

yarn android || npm run android

for ios

yarn ios || npm run ios
```
